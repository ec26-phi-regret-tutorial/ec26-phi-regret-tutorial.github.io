1.1 The ellipsoid against hope algorithm: solving the correlated equilibrium LP in multi-player games

The techniques we have covered based on regret minimization lead to a running time complexity growing polynomially in \(\frac{1}{\epsilon}\), where \(\epsilon > 0\) is the approximation quality of the equilibrium—be it a Nash equilibrium in zero-sum games or a (coarse) correlated equilibrium in general-sum games. Specifically, if we employ algorithms with (\(\Phi\)-)regret bounded by \(\sqrt{T}\) as a function of the time horizon, we need \(\frac{1}{\epsilon^2}\) iterations to reach an equilibrium. Can we do better in the regime where \(\epsilon << 1\)? The other main approach we have introduced for equilibrium computation is based on linear programming, which has the advantage of finding an exact equilibrium. But, as we saw in the last lecture, the LP describing (C)CEs has a number of variables that scales exponentially with the number of players.

We will now introduce ellipsoid against hope (EAH), a polynomial-time algorithm for computing (C)CEs even in multi-player games. It was first introduced by Papadimitriou and Roughgarden [PR08][PR08] C. H. Papadimitriou and T. Roughgarden. (2008). Computing correlated equilibria in multi-player games. Journal of the ACM.; in what follows, we mostly follow the generalized version of the algorithm due to Farina and Pipis [FP24][FP24] G. Farina and C. Pipis. (2024). Polynomial-Time Computation of Exact $Phi$-Equilibria in Polyhedral Games. Neural Information Processing Systems.. To be clear, this algorithm works in the centralized model, and it’s not compatible with the framework of online learning, although similarities do exist between the two approaches as we shall see.

Our goal is to compute an \(\epsilon\)-\(\Phi\)-equilibrium of a multilinear game; that is, a correlated distribution \(\mu \in \Delta\left(\mathcal{X}_1 \times \cdots \times \mathcal{X}_n\right)\) such that for any player \(i \in \left[n\right]\) and deviation function \(\phi_i \in \Phi_i\) mapping \(\mathcal{X}_i \to \mathcal{X}_i\),

\[\mathbb{E}_{\left(\mathbf{x}_1, \dots, \mathbf{x}_n\right) \sim \mu} u_i \left(\mathbf{x}_1, \dots, \mathbf{x}_n\right) \ge \mathbb{E}_{\left(\mathbf{x}_1, \dots, \mathbf{x}_n\right) \sim \mu} u_i \left(\phi_i \left(\mathbf{x}_i\right), \mathbf{x}_{-i}\right) - \epsilon.\](1)

The ellipsoid against hope framework. From a more abstract point of view, EAH deals with optimization problems of the form11We caution that \(\mathcal{X}\) here corresponds to \(\mathcal{X}_1 \times \cdots \times \mathcal{X}_n\) in the context of (1), even though we usually take \(\mathcal{X}\) to be the strategy set of a single player in the context of regret minimization.

\[\mathrm{find\ } \mu \in \Delta\left(\mathcal{X}\right) \mathrm{\ such\ that\ } \mathbb{E}_{\mathbf{x} \sim \mu} \left\langle \mathbf{y}, G\left(\mathbf{x}\right)\right\rangle \ge 0 \quad \forall \mathbf{y} \in \mathcal{Y},\](2)

where \(\mathcal{X} \subseteq \mathbb{R}^d, \mathcal{Y} \subseteq \mathbb{R}^k\), and \(G : \mathcal{X} \to \mathbb{R}^k\) is a function that can be evaluated efficiently. The crux in the optimization problem (2) lies in the fact that \(\mu\) resides in a high-dimensional space; a canonical case to think about is when \(\mathcal{X} = \mathcal{A}_1 \times \cdots \times \mathcal{A}_n\) in a normal-form game, so that even describing a distribution \(\mu\) could require specifying \(\prod_{i=1}^n |\mathcal{A}_i| - 1\) coordinates. We assume that \(\mathcal{Y}\), which corresponds to the set of deviations, admits a separation oracle; under mild geometric assumptions, it’s equivalent to posit merely a membership oracle, which returns whether a point \(\mathbf{y} \in \mathbb{R}^k\) belongs to \(\mathcal{Y}\) or not. A special case is when \(\mathcal{Y}\) is a polytope described with a polynomial number of constraints, as is the case for swap regret in normal-form games—each player’s set of deviations amounts to the set of stochastic matrices. The key assumption in the EAH framework [FP24][FP24] G. Farina and C. Pipis. (2024). Polynomial-Time Computation of Exact $Phi$-Equilibria in Polyhedral Games. Neural Information Processing Systems. is the admission of a good-enough-response (GER) oracle, which, given any \(\mathbf{y} \in \mathcal{Y}\), returns a point \(\mathbf{x} \in \mathcal{X}\) such that \(\left\langle \mathbf{y}, G\left(\mathbf{x}\right)\right\rangle \ge 0\).

The EAH algorithm enables solving (2) with just a separation oracle for \(\mathcal{Y}\) and a GER oracle. The basic idea is to consider an \(\epsilon\)-approximate version of the dual of (2),

\[\mathrm{find\ } \mathbf{y} \in \mathcal{Y} \mathrm{\ such\ that\ } \left\langle \mathbf{y}, G\left(x\right)\right\rangle \le - \epsilon \quad \forall \mathbf{x} \in \mathcal{X}.\](3)

On account of the fact that a GER oracle exists, (3) is guaranteed to be infeasible. Even so, EAH proceeds by executing the ellipsoid algorithm on that infeasible program—this is where the apt name “ellipsoid against hope” comes from. In every step \(t\) of the algorithm, we have a candidate \(\mathbf{y}^{\left(t\right)} \in \mathcal{Y}\), and use the GER oracle to produce a point \(x^{\left(t\right)} \in \mathcal{X}\) that refutes \(\mathbf{y}^{\left(t\right)}\); in fact, an entire halfspace in \(\mathbb{R}^k\). This goes on until the volume of the ellipsoid has shrank to a small enough amount. It then follows that

\[\forall \mathbf{y} \in \mathcal{Y} \exists t \in \left[T\right] \mathrm{\ such\ that\ } \left\langle \mathbf{y}, G\left(\mathbf{x}^{\left(t\right)}\right)\right\rangle > - \epsilon.\](4)

Thus,

\[\min_{\mathbf{y} \in \mathcal{Y}} \max_{\mu \in \Delta\left(\left[T\right]\right)} \sum_{t=1}^T \mu^{\left(t\right)} \left\langle \mathbf{y}, G\left(\mathbf{x}^{\left(t\right)}\right)\right\rangle > - \epsilon.\](5)

By the minimax theorem, we conclude that

\[\max_{\mu \in \Delta\left(\left[T\right]\right)} \min_{\mathbf{y} \in \mathcal{Y}} \sum_{t=1}^T \mu^{\left(t\right)} \left\langle \mathbf{y}, G\left(\mathbf{x}^{\left(t\right)}\right)\right\rangle > - \epsilon.\](6)

In other words, there is a convex combination of \(\mathbf{x}^{\left(1\right)}, \dots, \mathbf{x}^{\left(T\right)}\) that refutes any possible \(y \in \mathcal{Y}\). That convex combination, which is a certificate of dual infeasibility, will thus be an approximate solution to (2). The key point is that the resulting zero-sum game in (6) is much smaller than the one we started with, and can be solved with standard LP techniques.

Theorem 1.1.  Assuming the existence of a separation oracle for \(\mathcal{Y}\) and a GER oracle, EAH runs in time \(\mathrm{poly}\left(d, k, \log\left(\frac{1}{\epsilon}\right)\right)\) and returns an \(\epsilon\)-approximate solution to (2).

Let’s now see to apply this algorithm to solve (1). We assume that each \(\Phi_i\) contains linear functions of the form \(\mathbf{x}_i \mapsto \mathbf{M}_i \mathbf{x}_i \in \mathcal{X}_i \subseteq \mathbb{R}^{m_i}\), where the set of valid matrices \(M_i\) is a polytope \(\mathcal{Y}_i\) with a polynomial number of variables and constraints. (For example, in the example covered in the last lecture pertaining to swap regret in normal-form games, the set of column stochastic matrices is a Cartesian product of probability simplices.) We assume further that \(\Phi_i\) contains the identity matrix. Then (1) can be expressed as

\[\mathbb{E}_{\left(\mathbf{x}_1, \dots, \mathbf{x}_n\right) \sim \mu} \left\langle \mathbf{I} - \mathbf{M}_i, u_i \left(\mathbf{x}_{-i}\right) \otimes \mathbf{x}_i\right\rangle \ge - \epsilon \quad \forall i \in \left[n\right], \mathbf{M}_i \in \mathcal{Y}_i,\](7)

since

\[\mathbb{E}_{\left(\mathbf{x}_1, \dots, \mathbf{x}_n\right) \sim \mu} \left\langle \mathbf{x}_i - \mathbf{M}_i \mathbf{x}_i, \mathbf{u}_i \left(\mathbf{x}_{-i}\right)\right\rangle = \mathbb{E}_{\left(\mathbf{x}_1, \dots, \mathbf{x}_n\right) \sim \mu} \left\langle \mathbf{I} - \mathbf{M}_i, \mathbf{u}_i \left(\mathbf{x}_{-i}\right) \otimes \mathbf{x}_i\right\rangle.\](8)

This indeed adheres to (2). What’s left is to prove that (7) admits a GER oracle. To do so, let’s consider any \(i \in \left[n\right]\) and deviation \(\mathbf{M}_i \in \mathcal{Y}_i\). Taking \(\mathbf{x}_i \in \mathcal{X}_i\) to be a fixed point of \(\mathbf{M}_i\), we have

\[\mathbb{E}_{\left(\mathbf{x}_1, \dots, \mathbf{x}_n\right) \sim \mu} \left\langle \mathbf{x}_i - \mathbf{M}_i \mathbf{x}_i, u_i \left(\mathbf{x}_{-i}\right)\right\rangle = 0,\](9)

as desired.

Theorem 1.2.  If for each player \(i \in \left[n\right]\) in a multilinear game the set of deviations \(\Phi_i\) admits a separation oracle, there is an algorithm polynomial in \(\log\left(\frac{1}{\epsilon}\right)\), \(n\), and \(m = \max_{1 \le i \le n} m_i\) that outputs an \(\epsilon\)-\(\Phi\)-equilibrium.

Overall, we find that EAH and the framework of \(\Phi\)-regret have several conceptual similarities. Namely, they both operate over the set of deviations, returning in each round a fixed point of the corresponding deviation. Further, the correlated distribution they output is given as a mixture of product distributions. This ensures that the correlated distribution admits an efficient, compact representation. The final similarity is that they both require a separation oracle for the set of deviations, which brings us to the final topic of this lecture.

1.2 Semi-separation oracle

The main requirement in both EAH and the framework of Gordon, Greenwald and Marks [GGM08][GGM08] G. J. Gordon, A. Greenwald and C. Marks. (2008). No-regret learning in convex games. International Conference on Machine Learning. is a separation oracle for the set of deviations. Indeed, focusing on minimizing external regret, a separation oracle is enough to efficiently implement an algorithm such as “FTRL”; conversely, any regret minimizer should be able to perform separation; otherwise, there would be a fixed utility exerting high regret.

However, even when \(\Phi\) contains only linear functions mapping \(\mathcal{X} \to \mathcal{X}\), it can be NP-hard to implement a membership oracle, as shown recently by Daskalakis, Farina, Fishelson, Pipis and Schneider [Das+25][Das+25] C. Daskalakis, G. Farina, M. Fishelson, C. Pipis and J. Schneider. (2025). Efficient Learning and Computation of Linear Correlated Equilibrium in General Convex Games. Symposium on Theory of Computing (STOC).. Does this mean that minimizing \(\Phi\)-regret or applying the EAH framework is intractable? Not so. While so far we have insisted on having both a separation oracle for the set of deviations and an expected fixed-point oracle (that is, a good-enough-response oracle), it turns out that it’s enough to implement their disjunction; this was coined a semi-separation oracle by Daskalakis, Farina, Fishelson, Pipis and Schneider [Das+25] (Definition 1.4).

We will introduce this notion in the general setting where the set \(\Phi\) comprises nonlinear functions expressed in the following form.

Definition 1.3.  Given a map \(\psi : \mathcal{X} \to \mathbb{R}^{k'}\), the set of deviations \(\Phi^\psi\) is the set of all maps \(\phi: \mathcal{X} \to \mathcal{X}\) that can be expressed as the matrix-vector product \(\mathbf{K}\left(\phi\right) \psi\left(\mathbf{x}\right) + \mathbf{c}\left(\phi\right)\) for some matrix \(\mathbf{K} \in \mathbb{R}^{d \times k'}\) and vector \(\mathbf{c} \in \mathbb{R}^d\). We denote by \(k = d \times k' + d\) the dimension of \(\left(\mathbf{K}, \mathbf{c}\right)\).

One can think of \(\psi\) as a feature mapping; a natural example captured by Definition 1.3 is low-degree polynomials, where \(\psi\) contains all \(\le \ell\)-wise products of entries in \(x\). Minimizing regret with respect to the set describing all valid \(\left(\mathbf{K}, \mathbf{c}\right)\) is daunting, but it turns out that this is not necessary. One can instead solve the following simpler problem; the reason why this is sufficient is not covered in these notes.

Definition 1.4 (Semi-separation).  In the semi-separation problem we are given a function \(\phi : \mathcal{X} \to \mathbb{R}^d\) and we have to compute

  • either an \(\epsilon\)-expected fixed point \(\mu \in \Delta\left(\mathcal{X}\right)\) of \(\phi\),
  • or a point \(\mathbf{x} \in \mathcal{X}\) such that \(\phi\left(\mathbf{x}\right) \notin \mathcal{X}\).

Implementing the semi-separation oracle is actually not hard once we have an algorithm for computing expected fixed points. One can basically run the same algorithm: if during the execution of the algorithm we identify a point such that \(\phi\left(\mathbf{x}\right) \notin \mathcal{X}\), we can terminate as we have a certificate that \(\phi\) is not a valid mapping. Otherwise, the algorithm will be able to find an expected fixed point of \(\phi\), even though \(\phi\) may not necessarily be a valid transformation. In other words, the basic idea in the recent framework of Daskalakis, Farina, Fishelson, Pipis and Schneider [Das+25][Das+25] C. Daskalakis, G. Farina, M. Fishelson, C. Pipis and J. Schneider. (2025). Efficient Learning and Computation of Linear Correlated Equilibrium in General Convex Games. Symposium on Theory of Computing (STOC). is to make allowance for functions that do not necessarily map \(\mathcal{X}\) to \(\mathcal{X}\). To conclude, we state the main implication for minimizing \(\Phi^{\psi}\)-regret per Definition 1.3.

Theorem 1.5.  There is an algorithm, based on EAH, that computes an \(\epsilon\)-\(\Phi^\psi\)-equilibrium of any multilinear \(n\)-player game and has complexity \(\mathrm{poly}\left(n, m, k, \log\left(\frac{1}{\epsilon}\right)\right)\). Furthermore, in the online learning setting, there is an efficient algorithm with regret bounded as \(\mathrm{poly}\left(m, k\right) \sqrt{T}\) after \(T\) rounds.
