In this chapter, we introduce the notion of profile swap regret, which lies in between linear swap regret and full swap regret. A key property of profile swap regret is that it guarantees non-manipulability—in a sense that will be formalized—against a dynamic optimizer. What’s more, there is an efficient algorithm for minimizing profile swap regret. This notion was recently introduced by Arunachaleswaran, Collina, Mansour, Mohri, Schneider and Sivan [Aru+25][Aru+25] E. R. Arunachaleswaran, N. Collina, Y. Mansour, M. Mohri, J. Schneider and B. Sivan. (2025). Swap Regret and Correlated Equilibria Beyond Normal-Form Games. Conference on Economics and Computation (EC)., who also provided the first efficient algorithm. In what follows, we present the approach of Anagnostides, Farina, Fishelson, Luo and Schneider [Ana+26][Ana+26] I. Anagnostides, G. Farina, M. Fishelson, H. Luo and J. Schneider. (2026). Swap Regret Minimization Through Response-Based Approachability. Conference on Learning Theory (COLT)., which in turn relies on the response-based approachability algorithm of Bernstein and Shimkin [BS15][BS15] A. Bernstein and N. Shimkin. (2015). Response-Based Approachability with Applications to Generalized No-Regret Problems. Journal of Machine Learning Research..

6.1 Setup

We operate in the usual online linear optimization setting. The learner picks strategies from a convex and compact strategy set \(\mathcal{X} \subset \mathbb{R}^d\). At every round \(t \in \left[T\right]\), the learner first selects a strategy \(\mathbf{x}^{(t)} \in \mathcal{X}\); the adversary selects a utility vector \(\mathbf{u}^{(t)} \in \mathcal{U} \subseteq \mathbb{R}^d\); then the learner receives utility \(\left\langle \mathbf{x}^{(t)}, \mathbf{u}^{(t)}\right\rangle\), and observes \(\mathbf{u}^{(t)}\) as feedback. We make the standard normalization assumption \(\left\lvert \left\langle \mathbf{x}, \mathbf{u}\right\rangle\right\rvert \le 1\) for all \(\mathbf{x} \in \mathcal{X}, \mathbf{u} \in \mathcal{U}.\) In what follows, for a utility vector \(\mathbf{u} \in \mathcal{U}\), we define the best-response map \(b(\mathbf{u}) \in \operatorname*{\operatorname{arg}\,max}_{\mathbf{x} \in \mathcal{X}} \left\langle \mathbf{x}, \mathbf{u}\right\rangle,\) with ties broken arbitrarily.

Linear swap regret. We first recall the benchmark of linear swap regret. An affine map \(\phi: \mathcal{X} \to \mathbb{R}^d\) is an endomorphism of \(\mathcal{X}\) if \(\phi(\mathcal{X}) \subseteq \mathcal{X}\). We denote by \(\mathrm{End}(\mathcal{X})\) the set of affine endomorphisms. We write each \(\phi \in \mathrm{End}(\mathcal{X})\) as \(\phi(\mathbf{x}) = \mathbf{M} \mathbf{x} + va.\) The linear swap regret of the learner after \(T\) rounds is

\[\mathrm{LinearSwapReg}^{\left(T\right)} \coloneqq \max_{\phi \in \mathrm{End}\left(\mathcal{X}\right)} \sum_{t=1}^T \left\langle \phi\left(\mathbf{x}^{\left(t\right)}\right), \mathbf{u}^{\left(t\right)}\right\rangle - \sum_{t=1}^T \left\langle \mathbf{x}^{\left(t\right)}, \mathbf{u}^{\left(t\right)}\right\rangle.\](1)

This is a natural relaxation of (full) swap regret: instead of allowing arbitrary maps, the performance of the learner is compared against affine endomorphisms. In this case, the algorithm of Gordon, Greenwald and Marks [GGM08][GGM08] G. J. Gordon, A. Greenwald and C. Marks. (2008). No-regret learning in convex games. International Conference on Machine Learning. reduces linear swap regret to external regret over the \(d(d+1)\)-dimensional set deviations. This implies an online algorithm with linear swap regret growing as \(O(d \sqrt{T})\), which is information-theoretically optimal [Ana+26][Ana+26] I. Anagnostides, G. Farina, M. Fishelson, H. Luo and J. Schneider. (2026). Swap Regret Minimization Through Response-Based Approachability. Conference on Learning Theory (COLT).. On the other hand, the algorithm of Gordon, Greenwald and Marks [GGM08] is computationally inefficient [Das+25][Das+25] C. Daskalakis, G. Farina, M. Fishelson, C. Pipis and J. Schneider. (2025). Efficient Learning and Computation of Linear Correlated Equilibrium in General Convex Games. Symposium on Theory of Computing (STOC).. This lecture will cover an approachability framework that circumvents this barrier.

Correlated strategy profiles. A basic object in the approachability formulation is the correlated strategy profile (CSP). Specifically, for a realized pair \((\mathbf{x}^{(t)}, \mathbf{u}^{(t)})\), we define \(\kappa^{(t)} \coloneqq (\mathbf{u}^{(t)} \otimes \mathbf{x}^{(t)}, \mathbf{u}^{(t)}) \in \mathbb{R}^{d \times (d+1)}.\) The average CSP is

\[\overline{\kappa}^{\left(T\right)} \coloneqq \frac{1}{T} \sum_{t=1}^T \kappa^{\left(t\right)}.\](2)

For a CSP \(\kappa = (\mathbf{K}, vq)\) and an affine map \(\phi(\mathbf{x})=\mathbf{M} \mathbf{x} + va\), we use the shorthand notation \(\left\langle (\mathbf{M}, va), \kappa\right\rangle \coloneqq \left\langle \mathbf{M}, \mathbf{K}\right\rangle + \left\langle va, vq\right\rangle.\) With this notation at hand, linear swap regret can be rewritten as

\[\mathrm{LinearSwapReg}^{\left(T\right)} = T \max_{\phi=\left(\mathbf{M},va\right) \in \mathrm{End}\left(\mathcal{X}\right)} \left\langle \left(\mathbf{M}, va\right) - \left(\mathbf{I}, 0\right), \overline{\kappa}^{\left(T\right)}\right\rangle.\](3)

Using this reformulation, we will shortly see how to reduce minimizing linear swap regret to a suitable approachability instance.

6.2 Profile swap regret and non-manipulability

We now define the notion of profile swap regret and discuss its non-manipulability properties. To do so, we define the instantaneous regret as \(R(\mathbf{x}, \mathbf{u}) \coloneqq \max_{\mathbf{x}^* \in \mathcal{X}} \left\langle \mathbf{x}^* - \mathbf{x}, \mathbf{u}\right\rangle.\) Equivalently, we can write \(R(\mathbf{x}, \mathbf{u}) = \left\langle b(\mathbf{u}) - \mathbf{x}, \mathbf{u}\right\rangle.\) A valid decomposition of \(\overline{\kappa}^{(T)}\) can be written as

\[\overline{\kappa}^{\left(T\right)} = \sum_{j=1}^m \lambda_j \left(\mathbf{u}_j \otimes \mathbf{x}_j\right),\](4)

where \(\lambda_j \ge 0\), \(\sum_{j=1}^m \lambda_j = 1\), \(\mathbf{x}_j \in \mathcal{X}\), and \(\mathbf{u}_j \in \mathcal{U}\).

Definition 6.1 (Profile swap regret).  The profile swap regret of a CSP \(\overline{\kappa}^{(T)}\) is

\[\mathrm{ProfileSwapReg}^{\left(T\right)}\left(\overline{\kappa}^{\left(T\right)}\right) \coloneqq T \min_{\left(\lambda_j, \mathbf{x}_j, \mathbf{u}_j\right)_j} \sum_j \lambda_j R\left(\mathbf{x}_j, \mathbf{u}_j\right),\](5)

where the minimum is over all valid convex decompositions of \(\overline{\kappa}^{(T)}\).

The definition is admittedly somewhat cumbersome, but as we shall see it unlocks certain powerful properties. Specifically, this is a good time to discuss the notion of non-manipulability due to Deng, Schneider and Sivan [DSS19][DSS19] Y. Deng, J. Schneider and B. Sivan. (2019). Strategizing against no-regret learners. Neural information processing systems (NeurIPS)..

6.2.1 Manipulability against a dynamic optimizer

We consider a two-player game between a learner and an optimizer. The learner follows some learning algorithm, whereas the optimizer may choose its actions strategically over time after observing how the learner updates. We let \(u_\ell\) be the learner’s utility function and \(\mathcal{X}\) its strategy set. Similarly, we let \(u_o\) be the optimizer’s utility function and \(\mathcal{Y}\) its strategy set.

The key relevant benchmark in this setting is the optimizer’s Stackelberg value, which is the best utility the optimizer can guarantee by committing to a fixed strategy \(\mathbf{y} \in \mathcal{Y}\), while the learner best-responds to that strategy (with ties broken in favor of the optimizer):

\[\mathrm{Stack}\left(u_o\right) \coloneqq \max_{\mathbf{y} \in \mathcal{Y}} \max_{\mathbf{x} \in \mathrm{BR}\left(\mathbf{y}\right)} u_o \left(\mathbf{x}, \mathbf{y}\right),\](6)

where \(\mathrm{BR}(\mathbf{y}) \coloneqq \operatorname*{\operatorname{arg}\,max}_{\mathbf{x} \in \mathcal{X}} u_\ell (\mathbf{x}, \mathbf{y}).\) The Stackelberg value is what the optimizer can guarantee by means of a static commitment. Informally speaking, a learning algorithm is said to be manipulable if the optimizer can do strictly better than its Stackelberg value by using a dynamic strategy: instead of playing a fixed strategy \(\mathbf{y}\), the optimizer chooses a sequence \(\mathbf{y}^{(1)}, \mathbf{y}^{(2)}, \dots\) so as to steer the learner’s future play toward more favorable outcomes.

Equivalently, even if the optimizer fully understands the learner’s algorithm and chooses its strategies dynamically to influence future play, it cannot asymptotically obtain more than what it could already obtain by committing to a fixed strategy.

A key observation crystallized by Arunachaleswaran, Collina, Mansour, Mohri, Schneider and Sivan [Aru+25][Aru+25] E. R. Arunachaleswaran, N. Collina, Y. Mansour, M. Mohri, J. Schneider and B. Sivan. (2025). Swap Regret and Correlated Equilibria Beyond Normal-Form Games. Conference on Economics and Computation (EC). is that a learning algorithm that has vanishing profile swap regret cannot be manipulated by a dynamic optimizer.

Proposition 6.2.  If a learning algorithm guarantees \(\mathrm{ProfileSwapReg}^{(T)} = o(T)\), it is asymptotically non-manipulable.

6.3 Reducing to an approachability problem

Having introduced and motivated the notion of profile swap regret, we now reduce it to an approachability instance. We define \(\mathcal{K} \coloneqq \operatorname{conv} \left\{ (\mathbf{u} \otimes \mathbf{x}, \mathbf{u}) : \mathbf{u} \in \mathcal{U}, \mathbf{x} \in \mathcal{X} \right\}\) and \(\mathcal{S} \coloneqq \operatorname{conv} \left\{ (\mathbf{u} \otimes b(\mathbf{u}), \mathbf{u}) : \mathbf{u} \in \mathcal{U} \right\}.\) We measure approachability loss by

\[\mathrm{AppLoss}^{\left(T\right)} \coloneqq \min_{\mathbf{s} \in \mathcal{S}} \left\lVert \overline{\kappa}^{\left(T\right)} - \mathbf{s}\right\rVert_F.\](7)

We first point out that minimizing linear swap regret reduces to this approachability instance.

Lemma 6.3.  For any time \(T \in \mathbb{N}\),

\[\mathrm{LinearSwapReg}^{\left(T\right)} \le 2 T \mathrm{AppLoss}^{\left(T\right)} \cdot \max_{\phi \in \mathrm{End}\left(\mathcal{X}\right)} \left\lVert \phi\right\rVert_F.\](8)

Proof.  Let \(\mathbf{s} \in \mathcal{S}\) be closest to \(\overline{\kappa}^{(T)}\). Since \(\mathbf{s}\) is, by definition, a convex combination of points of the form \((\mathbf{u} \otimes b(\mathbf{u}), \mathbf{u})\) and \(b(\mathbf{u})\) maximizes \(\left\langle \cdot, \mathbf{u}\right\rangle\) over \(\mathcal{X}\), every endomorphism \(\phi \in \mathrm{End}(\mathcal{X})\) satisfies

\[\left\langle \phi - \left(\mathbf{I}, 0\right), \mathbf{s}\right\rangle \le 0.\](9)

Therefore, using (3),

\[\mathrm{LinearSwapReg}^{\left(T\right)} \le T \max_{\phi \in \mathrm{End}\left(\mathcal{X}\right)} \left\langle \phi-\left(\mathbf{I},0\right), \overline{\kappa}^{\left(T\right)} - \mathbf{s}\right\rangle.\](10)

The claim now follows from Cauchy-Schwarz.

What’s more, the same target set also captures profile swap regret. The following lemma follows essentially by the definition of profile swap regret.

Lemma 6.4.  The profile swap distance of \(\overline{\kappa}^{(T)}\) is equal to \(\mathrm{AppLoss}^{(T)}\).

The profile swap distance of a CSP \(\overline{\kappa}^{(T)}\) is infimum Euclidean distance of \(\overline{\kappa}^{(T)}\) from a CSP with zero profile swap regret. As a result, we conclude that minimizing the approachability loss simultaneously circumscribes both linear swap regret and profile swap regret.

6.4 Response-based approachability

The key observation is that the induced approachability problem can be solved efficiently through the response-based framework of Bernstein and Shimkin [BS15][BS15] A. Bernstein and N. Shimkin. (2015). Response-Based Approachability with Applications to Generalized No-Regret Problems. Journal of Machine Learning Research.; this was observed by Anagnostides, Farina, Fishelson, Luo and Schneider [Ana+26][Ana+26] I. Anagnostides, G. Farina, M. Fishelson, H. Luo and J. Schneider. (2026). Swap Regret Minimization Through Response-Based Approachability. Conference on Learning Theory (COLT)..

The algorithm proceeds by maintaining an accumulated displacement

\[\mathbf{U}^{\left(t\right)} \coloneqq \sum_{\tau=1}^t \left(\kappa^{\left(\tau\right)} - \mathbf{s}^{\left(\tau\right)}\right) \in \mathbb{R}^{d \times \left(d+1\right)}.\](11)

At every round \(t\), it solves the bilinear zero-sum game

\[\max_{\mathbf{x} \in \mathcal{X}} \min_{\mathbf{u} \in \mathcal{U}} \left\langle \mathbf{U}^{\left(t-1\right)}, \left(\mathbf{u} \otimes \mathbf{x}, \mathbf{u}\right)\right\rangle.\](12)

Let \((\mathbf{x}^{(t)}, \mathbf{u}_*^{(t)})\) be a pair of minimax strategies for this game. The algorithm then sets

\[\mathbf{s}^{\left(t\right)} \coloneqq \left(\mathbf{u}_*^{\left(t\right)} \otimes b\left(\mathbf{u}_*^{\left(t\right)}\right), \mathbf{u}_*^{\left(t\right)}\right) \in \mathcal{S}.\](13)

Finally, it plays the strategy \(\mathbf{x}^{(t)}\), whereupon it observes the utility \(\mathbf{u}^{(t)}\), forms the induced \(\kappa^{(t)}\), and updates the cumulative displacement \(\mathbf{U}^{(t)}\).

This response-based approachability algorithm is summarized below. All steps can be efficiently implemented through oracle access to \(\mathcal{X}\).

Algorithm 1 ([BS15]: Response-based approachability)
Input: Horizon \(T\), sets \(\mathcal{X}, \mathcal{U}\), best-response map \(b : \mathcal{U} \to \mathcal{X}\)
Initialize \(\mathbf{U}^{(0)} \coloneqq 0 \in \mathbb{R}^{d \times (d+1)}\)
for each round \(t = 1, \dots, T\) do
Compute maximin strategies \((\mathbf{x}^{(t)}, \mathbf{u}_*^{(t)})\) for
\[\max_{\mathbf{x} \in \mathcal{X}} \min_{\mathbf{u} \in \mathcal{U}} \left\langle \mathbf{U}^{\left(t-1\right)}, \left(\mathbf{u} \otimes \mathbf{x}, \mathbf{u}\right)\right\rangle\]
Set \(\mathbf{s}^{(t)} \coloneqq (\mathbf{u}_*^{(t)} \otimes b(\mathbf{u}_*^{(t)}), \mathbf{u}_*^{(t)})\)
Play \(\mathbf{x}^{(t)}\) and observe \(\mathbf{u}^{(t)}\)
Set \(\kappa^{(t)} \coloneqq (\mathbf{u}^{(t)} \otimes \mathbf{x}^{(t)}, \mathbf{u}^{(t)})\)
Update \(\mathbf{U}^{(t)} \coloneqq \mathbf{U}^{(t-1)} + \kappa^{(t)} - \mathbf{s}^{(t)}\)

The proof of correctness crucially relies on the minimax theorem, as we formalize below.

Lemma 6.5.  For every utility \(\mathbf{u}^{(t)} \in \mathcal{U}\), we have

\[\left\langle \mathbf{U}^{\left(t-1\right)}, \kappa^{\left(t\right)} - \mathbf{s}^{\left(t\right)}\right\rangle \ge 0.\](14)

Proof.  Let

\[g_t \left(\mathbf{x}, \mathbf{u}\right) \coloneqq \left\langle \mathbf{U}^{\left(t-1\right)}, \left(\mathbf{u} \otimes \mathbf{x}, \mathbf{u}\right)\right\rangle.\](15)

By the minimax theorem,

\[\max_{\mathbf{x} \in \mathcal{X}} \min_{\mathbf{u} \in \mathcal{U}} g_t \left(\mathbf{x}, \mathbf{u}\right) = \min_{\mathbf{u} \in \mathcal{U}} \max_{\mathbf{x} \in \mathcal{X}} g_t \left(\mathbf{x}, \mathbf{u}\right).\](16)

Thus,

\[g_t \left(\mathbf{x}^{\left(t\right)}, \mathbf{u}^{\left(t\right)}\right) \ge \min_{\mathbf{u} \in \mathcal{U}} \max_{\mathbf{x} \in \mathcal{X}} g_t \left(\mathbf{x}, \mathbf{u}\right).\](17)

Since \(\mathbf{u}_*^{(t)}\) is minimax, and \(b(\mathbf{u}_*^{(t)})\) is a feasible strategy in \(\mathcal{X}\),

\[\min_{\mathbf{u} \in \mathcal{U}} \max_{\mathbf{x} \in \mathcal{X}} g_t \left(\mathbf{x}, \mathbf{u}\right) \ge g_t \left(b\left(\mathbf{u}_*^{\left(t\right)}\right), \mathbf{u}_*^{\left(t\right)}\right).\](18)

Combining the two inequalities gives

\[\left\langle \mathbf{U}^{\left(t-1\right)}, \kappa^{\left(t\right)}\right\rangle \ge \left\langle \mathbf{U}^{\left(t-1\right)}, \mathbf{s}^{\left(t\right)}\right\rangle,\](19)

which is exactly (14).

The invariant implies approachability by the usual Pythagorean argument.

Lemma 6.6 (Pythagorean lemma).  Let \(\mathbf{z}^{(1)}, \dots, \mathbf{z}^{(T)}\) be vectors with \(\left\lVert \mathbf{z}^{(t)}\right\rVert_2 \le B\) for all \(t\). If

\[\left\langle \sum_{\tau=1}^{t-1} \mathbf{z}^{\left(\tau\right)}, \mathbf{z}^{\left(t\right)}\right\rangle \le 0\](20)

for every \(t\), then

\[\left\lVert \sum_{t=1}^T \mathbf{z}^{\left(t\right)}\right\rVert_2 \le B \sqrt{T}.\](21)

Applying Lemma 6.6 with \(\mathbf{z}^{(t)} = \mathbf{s}^{(t)} - \kappa^{(t)}\) gives

\[\mathrm{AppLoss}^{\left(T\right)} \le \left\lVert \frac{1}{T} \sum_{t=1}^T \left(\kappa^{\left(t\right)} - \mathbf{s}^{\left(t\right)}\right)\right\rVert_F \le \frac{2 B}{\sqrt{T}},\](22)

where \(B \coloneqq \max_{\mathbf{x} \in \mathcal{X}, \mathbf{u} \in \mathcal{U}} \left\lVert (\mathbf{u} \otimes \mathbf{x}, \mathbf{u})\right\rVert_F.\) We thus arrive at the following theorem.

Theorem 6.7.  For any sequence of utilities \(\mathbf{u}^{(1)}, \dots, \mathbf{u}^{(T)} \in \mathcal{U}\), the response-based approachability algorithm guarantees

\[\mathrm{LinearSwapReg}^{\left(T\right)} \le 4 \sqrt{T} \left(\max_{\mathbf{x} \in \mathcal{X}, \mathbf{u} \in \mathcal{U}} \left\lVert \mathbf{u}\right\rVert_2 \sqrt{\left\lVert \mathbf{x}\right\rVert_2^2 + 1}\right) \left(\max_{\phi \in \mathrm{End}\left(\mathcal{X}\right)} \left\lVert \phi\right\rVert_F\right),\](23)

and

\[\mathrm{ProfileSwapDist}^{\left(T\right)} \le \frac{2}{\sqrt{T}} \left(\max_{\mathbf{x} \in \mathcal{X}, \mathbf{u} \in \mathcal{U}} \left\lVert \mathbf{u}\right\rVert_2 \sqrt{\left\lVert \mathbf{x}\right\rVert_2^2 + 1}\right).\](24)

Through suitable preconditioning, it can be shown that this algorithm guarantees optimal linear swap regret [Ana+26][Ana+26] I. Anagnostides, G. Farina, M. Fishelson, H. Luo and J. Schneider. (2026). Swap Regret Minimization Through Response-Based Approachability. Conference on Learning Theory (COLT).. We further remark that this response-based framework can be extended to nonlinear deviations; we refer the interested reader to Anagnostides, Farina, Fishelson, Luo and Schneider [Ana+26].
